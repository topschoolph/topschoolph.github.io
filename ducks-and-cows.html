Answer:

1. 60:48 or 5:4

2. 48:108 or 4:9

Step-by-step explanation:

1. Remember that a duck has two legs and a cow has four legs. Then, there are 30 x 2 = 60 duck legs and 12 x 4 = 48 cow legs. Thus, the ratio of the number of duck legs to the number of cow legs is 60:48 which can be simplified to 5:4.

2. It is known that the total number of cow legs is 48, then the total number of duck and cow legs is 60 + 48 = 108. Therefore, the ratio of the total number of cow legs to the number of duck and cow legs is 48:108 which can be simplified to 4:9.

14. Grandpa has some cows and ducks in his farm. All in all thereare 8 heads and 26 legs, how many cows and ducks are there?STEPSUNDERSTAND:________PLAN:________SOLVE:No. ofCows' legsNo. ofDuckslegsFINAL ANSWER___________​


System of Equations

We let [tex]c[/tex] be as the number of cows.

We let [tex]d[/tex] be as the number of ducks.

From the information you have given, the combined number of heads of both ducks and cows is [tex]8[/tex], this tells me that there [tex]8[/tex] animals in Grandpa's farm. Therefore I can claim that [tex]c + d = 8[/tex].

A duck has [tex]2[/tex] legs making the number of legs of all duck in the farm is [tex]2d[/tex]. A cow has [tex]4[/tex] legs making the number of legs of all cows in the farm is [tex]4c[/tex]. The information also tells us that the combined number of legs of all the animals in Grandpa's farm is [tex]26[/tex] therefore I can claim this equation: [tex]2d +4c = 26[/tex]

Now we have two equations with two unknowns we can finally solve for them.

Manipulating the first Equation:

[tex]c + d = 8 \\ 2( c + d) =( 8)2 \\ 2c +2d = 16 [/tex]

Solving by Elimination:

[tex]2d +4c = 26 \\ 2d +4c -(2d +2c)= 26 -16 \\ 2c = 10 \\ c = 5[/tex]

There are [tex]5[/tex] cows! All we have to do now is to find the number of ducks from this equation: [tex]c + d = 8[/tex].

[tex]c + d = 8 \\ 5 +d = 8 \\ d = 3[/tex]

There are [tex]3[/tex] ducks!

22. 15. Given E-pig, goat, cow, horse) and F - (chicken, duck), find E-F. a. pig, goat, cow, horse, chicken, duck) b. (pig, goat, cow, horse) c. (chicken, duck) d. { }​


Answer:

The correct answer is B. E – F = {pig, goat, cow, horse}

Step-by-step explanation:

THE DIFFERENCE OF TWO SETS

The difference of two sets, written E – F is the set of all elements of E that are not elements of F.

Set E = {pig, goat, cow, horse}Set F = {chicken, duck}

Since there are no common elements in Set E and Set F, then we will not eliminate any of the elements in Set E. Therefore, the elements in Set E will remain the same.

ANSWER: E – F = {pig, goat, cow, horse}

                Also, F – E = {chicken, duck}

ADDITIONAL INFORMATION:

For cases where there are common elements in two sets, then we will eliminate those common elements.

Example: Let A = {1, 2, 3} and B = {3, 4, 5}

A – B = {1, 2}

Explanation: A – B is the set of all elements in A that do not belongs to B.

B – A = {4, 5}

Explanation: Similarly, B – A is the set of all elements in B that do not belongs to A.
