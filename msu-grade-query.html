11. Which of the following is NOT a component of Query Design View? Object Relationship Pane Run Query Design View Field


Field is not a component of Query Design View.

Query Design View is a view in Microsoft Access that allows you to design a query by specifying the tables and fields to include in the query, as well as any criteria to use to filter the data. The components of Query Design View include:

Object Relationship Pane: This is a graphical representation of the relationships between the tables in your database. You can use this pane to view and edit the relationships between tables.

Run Query: This button allows you to run the query and see the results.

Design View: This is the main area of the Query Design View where you can specify the tables and fields to include in the query, as well as any criteria to use to filter the data.

So, Field is not a component of Query Design View. It is a term used to refer to a column in a table or a field in a query.
