1. M B. Given AROS = ATXS, answer the following. T R S O X 6. RO 7.XS = 8. If RT = 30 cm, what is ST? 9. If RO = 4 + x and TX = 5 what is x? 10. If OS = 3x - 5 and XS = -2x, what is x?​


Answer:

Since AROS = ATXS, we can use this equation to solve the following problems:

6. RO = 7.XS

If we substitute RO = 7.XS into AROS = ATXS, we get:

AROS = A(7.XS) = 7.ATXS

This means that the area of triangle ROS is 7 times the area of triangle TXS. Since they share a height (RT), we can set their bases equal to each other:

0.5(RT)(ST) = 0.5(RT)(5)

ST = 5

Therefore, if RT = 30 cm, then ST = 5 cm.

7. RO = 4 + x and TX = 5

If we substitute RO = 4 + x and TX = 5 into AROS = ATXS, we get:

AROS = A(4 + x) = ATXS(5)

This means that the area of triangle ROS is 5 times the area of triangle TXS. Since they share a height (RT), we can set their bases equal to each other:

0.5(RT)(4 + x) = 0.5(RT)(5)

2 + 0.5x = 2.5

0.5x = 0.5

x = 1

Therefore, if RO = 4 + x and TX = 5, then x = 1.

10. OS = 3x - 5 and XS = -2x

If we substitute OS = 3x - 5 and XS = -2x into AROS = ATXS, we get:

AROS = A(3x - 5) = ATXS(-2x)

This means that the area of triangle ROS is -2 times the area of triangle TXS. Since they share a height (RT), we can set their bases equal to each other:

0.5(RT)(3x - 5) = 0.5(RT)(-2x)

1.5x - 2.5 = -x

2.5x = 2.5

x = 1

Therefore, if OS = 3x - 5 and XS = -2x, then x = 1.
