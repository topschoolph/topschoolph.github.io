16. measuring method for offset bend? ​


Answer:

(Offset Distance) X (Constant Multiplier) = Second Mark Distance (Distance between Marks). Example: 6” X 1.4 = 8.4” between 1st & 2nd mark. This calculated value is how far apart to make your marks from each other on the conduit and where to make your 45° bends.


17. Discuss the ffg.1. Explain the step on procedure in bending of:A. Elbow on 90°B. Offset BendC. Three Point SaddleD. U-bendE. 45° BendF. Z-bendG. End Coupling 2. What are the advantage of PVC conduit in electrical wiring?​


Explanation:

Elbow on 90°: To create a 90° elbow bend, the pipe is heated until it is malleable and then bent into a quarter circle shape using a bending machine or a press.

B. Offset Bend: To create an offset bend, the pipe is heated and bent in such a way that it creates an offset, or a "kink," in the pipe. This can be done using a bending machine or a press.

C. Three Point Saddle: To create a three point saddle bend, the pipe is heated and then bent over a saddle-shaped form using a bending machine or a press. This creates a bend with a "saddle" shape that allows the pipe to be mounted on a flat surface.

D. U-bend: To create a U-bend, the pipe is heated and then bent into a U shape using a bending machine or a press.

E. 45° Bend: To create a 45° bend, the pipe is heated and then bent at a 45° angle using a bending machine or a press.

F. Z-bend: To create a Z-bend, the pipe is heated and then bent into a Z shape using a bending machine or a press.

G. End Coupling: An end coupling is created by connecting two pipes together using a coupling, which is a short piece of pipe with threads on both ends. The pipes are first cleaned and then the coupling is attached to the ends of the pipes using a pipe wrench or other appropriate tool.
