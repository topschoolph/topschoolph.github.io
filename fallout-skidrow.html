5. 1.) Strontium-90 is a nuclide found in the fallout from an atomic bomb explosion. It is particularly harmful because it is similar to calcium in size and charge, so it may end up mixed with the calcium phosphate in cow,s milk. Strontium-90 undergoes beta decay to yield Yttrium-90. The half-life of this process is 28 years. What percentage of the Strontium-90 is left after exactly 56 years?​


Given:

[tex]t_{1/2} = \text{28 y}[/tex]

[tex]t = \text{56 y}[/tex]

Unknown:

[tex]\frac{m_{f}}{m_{i}}[/tex]

Solution:

Step 1: Calculate the rate constant.

[tex]k = \frac{ln(2)}{t_{1/2}}[/tex]

[tex]k = \frac{ln(2)}{\text{28 y}}[/tex]

[tex]k = 0.02475526 \: \text{/y}[/tex]

Step 2: Calculate the percentage of Sr-90 left.

[tex]ln(\frac{m_{f}}{m_{i}}) = -kt[/tex]

[tex]\frac{m_{f}}{m_{i}} = e^{-kt}[/tex]

[tex]\frac{m_{f}}{m_{i}} = e^{-\text{(0.02475526 /y)(56 y)}}[/tex]

[tex]\frac{m_{f}}{m_{i}} = 0.25[/tex]

[tex]\boxed{\frac{m_{f}}{m_{i}} = 25\%}[/tex]
