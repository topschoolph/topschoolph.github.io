Given:

Diameter = 45ft

45/2 = 22.5

Radius = 22.5ft

Radius = 6.858m

(Angular displacement) θ=?

θ = 60° x (2πrad/360°)

θ = 1.04rad

In case, (S=arc length, θ=angular displacement, R=radius)

S = θR

S = (1.04rad) x (6.858m)

20. A wheel 80cm in diameter turns at 120 rev/min. What is the angular velocity of the wheel in rad/s?


The angular velocity of the wheel : 4π rad/s

Further explanation

The motion of objects passing through a circular path will have linear velocity and angular velocity  

The linear velocity of an object is the velocity that an object has when moving in a circular direction in the direction of its rotation  

The length of the path the object is traveling around the circle:  

C = 2πr  

Linear speeds can be formulated:  

 v = 2πr/T

r = radius (m)  

T = period (s)  

The speed of the rotating object is called Angular velocity. This value is part of circular motion.  

The angular velocity is denoted by ω including the vector quantity. The unit used is usually rpm (rotation per minute) or radians / sec  

Relationship between angular and linear velocity

ω = v/r

Because the question is the angular velocity, then we will only convert rpm (rev / min)  in rad / s

A wheel turns at 120 rev/min, so

ω = 120 rpm

[tex]\rm \omega=2\pi\times(RPM)/60\\\\\omega=2\pi\times 120/60\\\\\omega=4\pi~rad/s[/tex]

30. A wheel with diameter 35 cm is rotating at 20 revolutions per minute. what is the speed of the rotating wheel in radians per second. This is the angular velocity of the wheel.


Since we are just being asked about giving the angular velocity (ω) of the rotating wheel in rad/s, we do not need to take into account the diameter of the wheel.

We only do that when the linear velocity is being asked. v=rω

Anyway.

So we are given with [tex]20 \frac{rev}{min} [/tex] and we want to make the unit as [tex] \frac{rad}{s} [/tex].

In order to do that, we just have to remember that 1 revolution=2π rad

Doing the necessary conversions:
[tex]20 \frac{rev}{min} * \frac{1 min}{60 s}* \frac{2 \pi rad}{1 rev} [/tex]

By doing the arithmetic you will be able to get the value of the angular velocity in rad/s.
