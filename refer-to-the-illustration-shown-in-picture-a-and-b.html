Rest and Motion Objects on the Picture

Objects are words that are often used to express something that is physically real. Generally, objects have a certain mass or weight. Objects are divided into two, namely inanimate objects and living things.

Living things or organisms are objects that have the characteristics of breathing, moving, adapting, multiplying, and needing food and water, while inanimate objects are the opposite. Inanimate objects can be distinguished by their form, namely liquid, solid, and gas.

Inanimate objects that are solid have a certain mass and cannot move if they are not influenced by external factors. An object at rest is a state in which the object remains in the same position. When at rest the velocity and velocity of an object are zero, while an object in motion is the state of an object that changes position. Objects in motion have speed and velocity and can even have acceleration.

Based on the explanation above, we can understand that:

Picture A is an image of a resting object because there are no external influencing factors.Picture B is an image of a motion object because there is an arrow which is an external factor. The external factor that is given to make an object move is a force, either a push or a pull.

So for question number 1, object A is a stationary object.Question number 2 is object B is a moving object.As for the answer to number 3, the object is stationary due to the absence of external factors that affect the object, such as a pull or push force, so that the object remains in the same position or does not experience displacement. Number 4, namely moving objects caused by external factors in the form of a visible thrust based on the existing arrows, causes the object to move and change its position from before.
