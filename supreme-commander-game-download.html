22. While waiting for a video game to download, you notice that 30% of 32,000 kilobytes have been downloaded so far. How many kilobytes have been downloaded so far


Answer:

9600 kilobytes have been downloaded so far.

Step-by-step explanation:

Finding the Percentage

Let us analyze the problem.

What is asked?

How many kilobytes have been downloaded so far?

What are the given?

total number of kilobytes of the game - 32000

kilobytes that have been downloaded so far - 30%

To find the amount of kilobytes that have been downloaded so far, we need to get the percentage or the 30% of 32000.

To do that we have the formula:

Percentage = Base × Rate

Percentage is the part of the base determined by the rate.

Base is the whole on which the rate operates. It is the number represents 100% or the total value of something or the whole thing.

Rate is the number of hundredths parts taken. It is the number with a percent sign (%).

Based on our given, the base is 32000 and the rate is 30%.

Solution:

Before getting the percentage, we need to convert the rate to decimal first.

Converting Percent to Decimal

We have two ways to convert percent to decimal.

Divide by 100.

Using a calculator is the easiest way to divide this but if there is no available calculator, do the long division.

30 ÷ 100 = 0.30

Move the decimal point.

Imagine that there is a decimal point after the last digit. Move the decimal point twice going to your left.

30. = 0.30

The rate is 0.30.

For example of converting percent to decimal, visit the links:

https://brainly.ph/question/1454095

https://brainly.ph/question/1892802

Now we can proceed with finding the percentage. Substitute the values.

Percentage = Base × Rate

= 32000 × 0.30

= 9600

Final Answer:

9600 kilobytes have been downloaded.

There are problems that instead of finding the percentage, we need to find the rate or the base. Below are the formula for each.

Finding the Rate

In finding the rate, the formula is:

Rate = Percentage / Base
