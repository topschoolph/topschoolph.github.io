A. 7/2 & 1

2x² + 7 = 9x?

2x² - 9x + 7 = 0

(2x - 7) (x - 1) = 0

2x - 7 = 0                     x - 1 = 0

2x - 7 + 7 = 0 + 7         x - 1 + 1 = 0 + 1

2x  = 7                         x = 1
