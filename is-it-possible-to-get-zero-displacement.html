1. it is possible to get zero displacement? when can displacement be equal to zero?​


question:

it is possible to get zero displacement?

answer:

• Displacement is a vector quantity, so we need to specify a magnitude and direction in order to characterize it fully. Unlike distance, which can only get more significant on any path taken, displacement can be positive, negative, or zero. Displacement is zero when you start and end at the same spot.

question:

• when can displacement be equal to zero?

answer:

When the final position and initial position of the object is same then displacement is said to be zero for example an object/body moving in a circular motion when reaches the starting point after completing one complete cirecle is said to have zero displacement.
