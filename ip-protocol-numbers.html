21. Define the following. If possible, provide examples. Write your answers on a sheet of paper. Please write legibly.IP (Internet Protocol) AddressIPv4 (Internet Protocol Version 4 (IPv4) IPv6 (Internet Protocol Version 6)Static IP AddressDynamic IP AddressSubnet mask​


Answer:

1. An IP address, or Internet Protocol address, is a series of numbers that identifies any device on a network. Computers use IP addresses to communicate with each other both over the internet as well as on other networks.

2. Internet Protocol Version 4 (IPv4) is the fourth revision of the Internet Protocol and a widely used protocol in data communication over different kinds of networks. IPv4 is a connectionless protocol used in packet-switched layer networks, such as Ethernet.

3. The Internet Protocol version 6 or IPv6 is a version of the Internet Protocol (IP). The Internet Protocol allows the transfer of data through a network to IP addresses (IPv4 or IPv6), which identify the different devices that are connected to the Internet, allowing communications among them.

4. When a device is assigned a static IP address, the address does not change. Most devices use dynamic IP addresses, which are assigned by the network when they connect and change over time.

5. A dynamic IP address is an IP address that an ISP lets you use temporarily. If a dynamic address is not in use, it can be automatically assigned to a different device. Dynamic IP addresses are assigned using either DHCP or PPPoE.

6. A subnet mask is a 32-bit number used to differentiate the network component of an IP address by dividing the IP address into a network address and host address. It does so with bit arithmetic whereby a network address is bit multiplied by the subnet mask reveal the underlying subnetwork.

30. in interact through internet protocol IP adresses​


Answer:

An Internet Protocol address (IP address) is a numerical label assigned to each device (e.g., computer, printer) participating in a computer network that uses the Internet Protocol for communication. An IP address serves two principal functions: host or network interface identification and location addressing. Its role has been characterized as follows: “A name indicates what we seek. An address indicates where it is. A route indicates how to get there.”
