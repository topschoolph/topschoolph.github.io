Graph of a Function

A function is a particular kind of relation that we described as one in which each x-coordinate corresponds to just one y-coordinate. We defined the notation required to work with functions, and described a function as a process, algebraically. The collection of points that satisfy the equation y = f is the graph of a function (x). That is, if and only if y = f, the point (x, y) is on the graph of f (x).

For instance, to plot f, we plot the formula y = f (x). In particular, we test for symmetry, look for intercepts, and plot extra points if necessary. We set y = 0 in order to find the x-intercepts. Given that y = f(x), f(x) must be zero.

Translation of a graph

On the left is the graph of the absolute value function. On the right is its translation to a "new origin" at (3, 4).

The equation of the absolute value function is

y = |x|.

The equation of its translation to (3, 4) is

y − 4 = |x − 3|.

For, when x = 3, then y − 4 = 0, that is, y = 4.

Thus the point (3, 4) is that point on the translated graph that was originally at (0, 0).

In general,

If the graph of

y = f(x)

is translated a units horizontally and b units

vertically, then the equation of the translated

graph is

y − b = f(x − a).

When f(x) is translated a units horizontally, then the argument of f(x) becomes x − a. In the example above, the argument of |x| becomes x − 3.

We will prove that below.

x − (−5) = x + 5.
